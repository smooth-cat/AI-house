## 接口说明文档
### 1. 登录请求
##### 描述：`发送登录请求，返回一个用户是否登录成功的json对象`
##### 请求方式：`POST`
##### 发送数据格式：`'Content-Type':'application/json'`
##### 发送参数： 
|参数名|类型|参数说明|
|:--|:--|:--|
|name|String|用户名|
|password|String|密码|
##### 返回参数：
|参数名|类型|参数说明|
|:--|:--|:--|
|msg|String|登录反馈信息|
|mark|Boolean|是否登录成功|
##### 返回参数示例：
```javascript
  {
    msg:'登录成功',
    mark:true
  }
```

### 3. 退出请求
##### 描述：`发送退出请求，返回空字符串`
##### 请求方式：`GET`
##### 发送参数： 
|参数名|类型|参数说明|
|:--|:--|:--|
|name|String|用户名|
##### 返回字符串：`'已退出'`

### 4. 获取温度、湿度请求
##### 描述：`获取实时的温度和湿度的请求`
##### 请求方式：`GET`
##### 发送参数： 
|参数名|类型|参数说明|
|:--|:--|:--|
|name|String|用户名|
##### 返回参数： 
|参数名|类型|参数说明|
|:--|:--|:--|
|temperature|Number|实时温度|
|damp|Number|实时湿度|
##### 返回参数示例：
```javascript
  {
    temperature:26.26,
    damp:50.32
  }
```
### 5. 获取空调、灯光设置请求
##### 描述：`获取上一次退出时空调和灯光设置的请求`
##### 请求方式：`GET`
##### 发送参数： 
|参数名|类型|参数说明|
|:--|:--|:--|
|name|String|用户名|
##### 返回参数： 
|参数名|类型|参数说明|
|:--|:--|:--|
|dinning_l|Boolean|厨房灯光|
|bath_l|Boolean|浴室灯光|
|sitting_l|Number|客厅灯光|
|bedroom_l|Number|卧室灯光|
|disable|Boolean|空调是否关闭|
|aim_temp|Number|空调目标温度|
|damp_dispel|String|除湿是否开启|
|mode|String|空调模式|
|speed|Number|空调风速|
##### 返回参数示例：
```javascript
  {
    dinning_l:true,
    bath_l:false,
    sitting_l:66,
    bedroom_l:77,
    aim_temp: 30,
    damp_dispel: "除湿已开启",
    mode: "送风",
    speed: 3
  }
```
